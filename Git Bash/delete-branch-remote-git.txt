### Delete a remote GIT branch
To delete a remote branch you can use the following command:

```git push <remote_name> --delete <branch_name>```

Example Test:

```
Narasepta@DESKTOP-K36IDGM MINGW64 /d/Bash & Config/Linux-Bash-Config (master)
$ git push origin --delete patch-1
To https://github.com/codedadu/Linux-Bash-Config.git
 - [deleted]         patch-1
```
